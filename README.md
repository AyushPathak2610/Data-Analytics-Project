## Prerequisites

Ensure you have the following libraries installed:

- numpy
- pandas
- matplotlib
- seaborn
- plotly
- squarify
- scipy

You can install these using pip:

```sh
pip install numpy pandas matplotlib seaborn plotly squarify scipy
